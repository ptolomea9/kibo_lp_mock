# Content Gaps: Recommended New Landing Pages

## Overview
This document identifies content gaps discovered during the keyword audit where high-value search terms lack optimal landing page destinations. Recommendations include new pages to build and content enhancements to existing pages.

---

## Priority 1: Critical Gaps (Build Immediately)

### 1. OMS Pricing Page
**Gap Identified:** Keyword `[order management system pricing]` has no landing page
**Current State:** No public pricing information available
**Search Intent:** Bottom-funnel, high purchase intent

**Recommendation:**
- Create `/platform/order-management/pricing/` or `/pricing/`
- Include pricing tier structure (even if "Contact for Quote" for enterprise)
- Add comparison table showing package differences
- Include ROI calculator or TCO comparison tool
- Feature Forrester TEI 167% ROI data prominently

**Alternative (Short-term):** Route to `/speak-with-an-expert/` with pricing-focused messaging

---

### 2. Dedicated OMS Comparison Page
**Gap Identified:** Keywords like `[best order management systems]` need comparison content
**Current State:** Product pages exist but no comparison-focused content
**Search Intent:** Mid-funnel consideration, competitive research

**Recommendation:**
- Create `/platform/order-management/compare/` or `/resources/oms-comparison/`
- Include feature comparison matrix vs. competitors (Manhattan, Blue Yonder, etc.)
- Highlight analyst recognition (Forrester Strong Performer, Gartner Challenger)
- Add customer testimonial quotes comparing to previous solutions
- Include "Why KIBO" section with key differentiators

---

### 3. B2B Mobile Commerce Content
**Gap Identified:** Keyword `[b2b mobile commerce]` lacks dedicated content
**Current State:** No explicit mobile commerce messaging on B2B pages
**Search Intent:** Feature-specific research

**Recommendation:**
- Add mobile commerce section to `/solutions/b2b/` with anchor `#mobile`
- Cover responsive B2B portals, mobile ordering capabilities
- Include mobile app integration options
- Feature mobile-first buyer experiences

---
